Madsjeez Seller Browser v0.1.0
===============================

INSTALACION (recomendada)
-------------------------
1. Ejecuta MadsjeezSellerBrowserSetup.exe
2. Sigue el asistente grafico (Siguiente > Instalar > Finalizar)
3. Se crea acceso directo en escritorio y menu Inicio
4. El navegador se abre automaticamente al terminar

REQUISITOS
----------
- Windows 10 u 11 (64 bits)
- .NET 8 Desktop Runtime (x64)
  Descarga: https://dotnet.microsoft.com/download/dotnet/8.0
  (busca "Desktop Runtime" para Windows x64)

DESINSTALAR
-----------
Panel de control > Agregar o quitar programas > Madsjeez Seller Browser

SMARTSCREEN
-----------
Si Windows muestra un aviso de seguridad, el instalador firmado
(SignPath Foundation) lo reduce. Mientras tanto: Mas informacion >
Ejecutar de todas formas.
